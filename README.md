# Stock-Trading-Simulator
Allows you to trade stocks in real time using a virtual currency.

Live data is retrived using the Yahoo Finance API.

Requirements

Version 1
- pandas_datareader
- yfinance
- matplotlib
- datetime

Version 2
- pandas_datareader
- yfinance
- matplotlib
- datetime
- hashlib
- sqlite3

Version 3
- pandas_datareader
- yfinance
- matplotlib
- datetime
- hashlib
- sqlite3
- kivy
- kivyMD
